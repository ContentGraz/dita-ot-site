DITA Open Toolkit 2.1

This document describes the DITA Open Toolkit project—what the project is, and how to use the site.

What is the DITA Open Toolkit?

The DITA Open Toolkit, or DITA-OT for short, is a set of Java-based, open-source tools that provide processing for DITA maps and topic content. You can download the toolkit and install it for free to get started with topic-based writing and publishing.

The DITA-OT is licensed under the Apache 2.0 open-source license.

Note:

While the DITA Standard itself is owned and developed by OASIS, the DITA Open Toolkit is an independent, open-source implementation of the standard.

Key output formats for the toolkit include:

  • PDF
  • HTML – HTML5 and XHTML
  • TocJS – HTML5 with a JavaScript frameset navigation
  • HTML Help – Microsoft compiled help (.chm)
  • Eclipse Help – HTML with navigation and index for Eclipse information centers
  • Eclipse Content – normalized DITA plus Eclipse project files
  • Java Help – HTML topics, table of contents and index for the Java Help platform
  • ODT – OpenDocument Text
  • Word RTF (with some limitations)
  • DocBook
  • troff
