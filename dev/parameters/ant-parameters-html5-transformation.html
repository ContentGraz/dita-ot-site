Ant parameters: HTML5

Certain parameters are specific to the HTML5 transformation.

nav-toc
Generate navigation TOC into topic pages The following values are supported:
  • none (default) – No TOC
  • partial – Partial TOC that show current topic
  • full – Full TOC
