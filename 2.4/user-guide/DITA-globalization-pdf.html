Supported languages: PDF transformations

The DITA Open Toolkit supports a slightly smaller set of languages for the PDF (pdf2) transformation. This transformation was donated to the DITA-OT project after the project inception, and it uses a different and larger set of generated text than the HTML-based transformations.
