Installing the client package

For the simplest installation experience, install the client package. This package can be installed on Linux, Mac OS X, and Windows. It contains everything that you need to run the DITA-OT except for Java.

Before you begin

  • Ensure that you have Java JRE or JDK, version 7 or later installed.
  • Ensure that you have HTML Help Workshop installed, if you want to generate HTML Help.

Procedure

  1. Download the client package from the project website at www.dita-ot.org.
    Operating system File name
    Linux or Mac OS X DITA-OT2.1.0-M1_client_bin.tar.gz
    Windows DITA-OT2.1.0-M1_client_bin.zip
  2. Extract the contents of the package to the directory where you want to install the DITA-OT.
  3. Optional: Add the absolute path for the bin directory to the PATH system variable.
    This defines the necessary environment variable to run the dita command from the command-line.
